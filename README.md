## 📜프로젝트 개요
"Let Me In" 프로젝트는 취업 면접에서 제출할 수 있는 포트폴리오 및 이력서를 위한 웹사이트 제작입니다. 이 사이트는 나의 기술과 프로젝트들을 깔끔하게 정리하여 보여줌으로써 개발 능력을 강조하는 데 목적이 있습니다. 직접 제작한 포트폴리오 웹사이트를 통해 디자인, 프론트엔드 기술을 동시에 피력하며, 사용자 친화적인 인터페이스와 반응형 웹 디자인을 구현할 계획입니다.
